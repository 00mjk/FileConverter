Description

File Converter is a very simple tool which allows you to convert one or several file(s) from one format to another using the context menu in windows explorer.

You can convert files from formats Aiff, Ape, Avi, Flac, M4a, Mp3, Mp4, Ogg, Wav, Wma to formats Mp3, Ogg, Wav, Flac.

The interface is designed to be as light as possible to keep the application pleasant to use.

You can heavily customize the application configuration to add or remove conversion presets into the files context menus.

I hope this application will help you :)

It is open source (GLP v3), free and without any adds so you just have to enjoy it!
